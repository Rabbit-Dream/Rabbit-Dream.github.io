Linux常用命令

这里只记录了常用的命令及其选项参数，具体可以查看新手Linux命令

常见Linux命令格式：命令名称 [命令参数] [命令对象]

常用系统工作命令

打印

  • echo -e 打印变量
    • -e：激活后面打印内容中转义字符

重启/开机

  • reboot 重启
  • shutdown 关机
  • shutdown -h now 立刻关机
  • shutdown -h +10 十分钟后关机
  • shutdown -h 20:50 20:50关机
  • init 0 关机
  • init 6 重启
  • poweroff 立刻关机
  • poweroff -f 强制关机
  • poweroff -i 关机之前关闭所有网络接口
  • poweroff -h 关机之前将系统中所有硬件设置为备用模式

下载

  • wget 用来从指定的URL下载文件(非常稳定)
  • wget url 下载单个文件
    • wget -c url 断点续传
    • wget --spider url 测试下载链接
    • wget -O 指定文件名 url 以指定文件名保存
    • wget -b url 后台下载
    • wget -i file.txt 多个文件下载（file.txt保存多个下载链接）
    • wget --mirror -p --convert-links -P ./LOCAL url
      • --miror 开户镜像下载
      • -p 下载所有为了html页面显示正常的文件
      • --convert-links 下载后，转换成本地的链接
      • -P ./LOCAL 保存所有文件和目录到本地指定目录
    • wget -r -A.后缀 url 下载指定格式文件（后缀：pdf/mp4/png/...）
    • wget ftp-url 匿名ftp下载
  • wget --ftp-user=USERNAME --ftp-password=PASSWORD url 用户名和密码认证ftp下载

查看/结束进程

  • ps 查看进程
    • 具体选项
    • ps -a 查看所有进程
    • ps -aux 产看所有进程及其用户
  • top 实时动态地查看系统的整体运行情况
  • htop 需安装：<sudo apt install htop>
    • top高级版,超级好用。
    • 快速查看关键性能统计数据，如CPU（多核布局）、内存/交换使用
    • 可以横向或纵向滚动浏览进程列表，以查看所有的进程和完整的命令行
    • 杀掉进程时可以直接选择而不需要输入进程号
    • 通过鼠标操作条目
    • 比top启动得更快
  • kill PID 结束进程
  • kill -9 PID 强制结束进程
  • pidof 进程名称 查找指定进程名称的进程id号
  • killall (选项) 进程名称 使用进程的名称来杀死进程（好用）
    • 具体选项

系统状态检测命令

  • ifconfig 配置和显示Linux内核中网络接口的网络参数。具体参数
  • unname -a 打印当前系统相关信息
  • uptime 打印系统总共运行了多长时间和系统的平均负载
  • uptime -V 显示指令的版本信息
  • free -b/k/m 显示当前系统未使用的和已使用的内存数目(以Byte/KB/M为单位)
  • who 显示目前登录系统的用户信息
  • last -n 用户最近登录情况（固定n行
  • history n 显示n个数目的历史命令
  • history !n 显示第n个历史命令

文本文件编辑命令

  • cat m1 在屏幕上显示文件ml的内容
  • cat m1 m2 同时显示文件ml和m2的内容
  • cat m1 m2 > file 将文件ml和m2合并后放入文件file中
  • more -dc file 屏幕上显示之前清屏，在最下方显示完成百分比（部分查看）
  • more -c -10 file 每10行显示一次
  • head -n file 显示文件开头n行（默认10行）
  • tail file 显示文件尾10行（默认）
  • tail +20 file 显示文件file的内容，从第20行至文件末尾
  • tail -c 10 file 显示文件file的最后10个字符
  • stat file 显示文件状态信息（比ls命令更详细）
  • stat -f file 显示文件系统状态而非文件状态
  • stat -t file 以简洁方式输出信息
  • cut 显示行中的指定部分，删除文件中指定字段
  • cut -f n file 显示file内的第n字段（第n列）
  • cut -fm,n file 显示file内的第n字段（第m,n列）
  • cut -fn --complement file 显示file内的除第n字段（第n列）之外的字段（列）
  • cut -f2 -d";" file -d指定字段分隔符
  • diff 比较给定的两个文件内容的不同或两个文件夹里既有相同文件名的文件具体选项参数

文件目录管理命令

  • touch file 新建文件

  • mkdir folder 新建文件夹

  • cp file folder/file1 将文件file复制到目录forlder下并改名为file1

  • cp file folder 将文件file复制到目录forlder下

  • cp -r folder1 folder2 将目录forlder1下所有文件复制到目录folder2下

  • cp -i folder1 m*.c forlder2 将目录forlder1下所有y以m打头的所有.c文件复制到目录folder2下

  • mv file1 file2 将文件file1改名为file2

  • mv file1 forlder 将文件file移动到目录forlder下

  • rm file 删除文件

  • rm -i file 删除之前询问用户

  • rm -r folder 递归删除目录forlder下所有文件

  • rm -f file 强制删除文件

  • rm -rf folder 强制删除目录folder

    rm使用一定要小心，一旦删除无法恢复

  • file 探测给定文件的类型

打包压缩与搜索命令

  • tar -cvf x.tar file 打包

  • tar -xvf x.tar 解包

  • tar -zcvf x.tar.gz file gz压缩

  • tar -zxvf x.tar.gz gz解压缩

  • tar -jcvf x.tar.bz2 file bz2压缩

  • tar -jxvf x.tar.bz2 bz2解压缩

  • zip x.zip file zip压缩

  • unzip x.zip zip解压缩

  • 解压缩到指定路径：

    • tar ....... -C 指定路径
    • zip/unzip -d 指定路径 x.zip

  • grep "content" file

  • grep -n "content" file 显示content在文件中的行数

  • grep -v "content" file 除了content其他都显示

  • grep "^content" file 显示以content开头

  • grep "content$" file 显示以content结尾

  • find ./-name filename 查找当前目录下所有名为filename的文件

  • find ./-name '*.sh' 查找当前目录下所有后缀为.sh的文件

  • find ./-name "[A-Z]" 查找当前目录下所有以大写字母开头的文件

  • find /tmp -size 2M 查找/tmp目录下等于2M的文件

  • find /tmp -size +2M 查找/tmp目录下大于2M的文件

  • find /tmp -size -2M 查找/tmp目录下小于2M的文件

  • find ./ -size +4k -size -5M 查找当前目录下大于4k，小于5M的文件

  • find ./ -perm 777 查找当前目录下权限为777的文件或目录

权限

字母法：

  • chmod u=权限,g=权限,o=权限 file/folder唯一设定权限

  • chmod u+权限,g+权限,o+权限 file/folder增加权限

  • chmod u-权限,g-权限,o-权限 file/folder增加权限

  • u文件拥有者
    g同组
    o其他

  • 权限：

    • r：表可读
    • w：表可写
    • x：表可执行

当权限一致可以和在一块写：

shell

数字法：

  • chomd ugo对应权限数字 file/folder 修改权限

    • r------>4
    • w------>2
    • x------->1
    shell

多用户

  • sudo useradd username -m 新建用户

  • sudo passwd username 设置修改密码

  • su username 切换用户

  • su - username 切换用户并进入家目录

  • whoami 查看用户名

  • ssh username@IP 远程操作别的电脑

  • who 查看谁在登录

  • userder username 删除用户

  • sudo userder -r username 删除用户并删除他的家目录

  • sudo -s 进入超级管理员

  • 给用户超级管理员权限：

    • sudo usermod -a -G adm username
    • sudo usermod -a -G sudo username

  • grouadd groupname 新建组

chmod、chown、chgrp的区别

  • chmod：用与设置修改文件变动的权限

  • chown：转变文件拥有者

    • chown username:usergroup file/folder

  • chgrp：转变文件所属用户组

    • chgrp usergroup file/folder

在这三个命令后面加-R参数表示递归整个目录
