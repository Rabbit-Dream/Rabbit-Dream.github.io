Git简易教程
发表于 2020-02-22| 更新于 2020-02-23|教程日常杂耍
字数总计:344|阅读时长: 1 分钟
| 阅读量:|评论数:

  • 安装后

    • git config --global user.name "Your Name"

      Code
      
git config --global user.email "email@example.com"

      Code
      
git reset --hard commit_id    # 通过`commit_id`穿梭到指定版本

      Code
      
   ssh-keygen -t rsa -C "youremail@example.com"
      • 这时.ssh下有id_rsa（私钥）id_rsa.pub（公钥）

    • 登录GitHub，打开Account setting

    • 找到SSH Keys，然后Add SSH Key填上任意Title，在Key文本框粘贴id_rsa.pub

  • 添加远程仓库

  • 根据提示：克隆新仓库或者把一个已有的本地仓库与之关联

  • 然后在本地仓库文件夹下打开Git Bash输入git push -u origin master

  • 如果你的本地文件夹为创建成仓库：

    • git init
    • git add .
    • git commit -m "new creat"
    • git push -u origin master
